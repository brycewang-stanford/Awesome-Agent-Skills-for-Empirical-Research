# marginaleffects - Built-in Package Skill

Source: https://github.com/vincentarelbundock/marginaleffects

The marginaleffects R/Python package includes a built-in SKILL.md for AI agents.
Covers predictions, comparisons, slopes, marginal effects, ATE/ATT/CATE,
hypothesis testing, contrasts, counterfactuals, risk ratios, odds ratios,
and causal inference with G-computation.
