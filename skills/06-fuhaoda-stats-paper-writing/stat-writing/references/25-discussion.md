# Discussion / Conclusion guidance

## Goal
Close the loop: return to the motivation from the Introduction and clearly state what we learned.

## Minimum checklist
1. **Summary of contributions**: restate the 2–5 main contributions (in plain language).
2. **Answer the motivating question**: what is the key takeaway?
3. **Comparison to prior work** (brief): how does this differ/improve?
4. **Limitations**: be honest, specific (assumptions, data, computation, generality).
5. **Future work**: realistic next steps.
6. **Practical implications**: if applied, what should practitioners do?

## Strong closing paragraph template
- “We developed … to address … Under … we show … In simulations and an application to … we find … The method is most useful when …”

## Don’ts
- Do not introduce entirely new results that were not shown earlier.
- Do not overclaim (“guaranteed”, “solves all”).

## Output format
Return a LaTeX-ready Discussion/Conclusion section with clear paragraphing.
