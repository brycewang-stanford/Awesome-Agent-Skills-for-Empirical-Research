## Pre-Modern CRDC (1968-2008)

### Original Purpose

The collection began in 1968 as the **Elementary and Secondary School Civil Rights Survey**:
- Created to monitor school desegregation
- Focused on race/ethnicity data
- Supported enforcement of Civil Rights Act of 1964

### Key Characteristics

| Attribute | Description |
|-----------|-------------|
| Name | Elementary and Secondary School Survey |
| Frequency | Irregular (not consistently biennial) |
| Coverage | Sampled districts |
| Focus | Desegregation, race data |
| Technology | Paper-based, later electronic |

### Limitations of Pre-Modern Data

- **Not comparable** to modern CRDC
- Different collection instruments
- Different variable definitions
- Different sampling approaches
- Limited electronic availability

## Modern CRDC Era (2009-Present)

### The 2009-10 Redesign

Major transformation of the collection:
- New collection system developed
- Expanded data elements
- Focus on educational opportunity
- NCES partnership for quality improvement

### Key Changes from Pre-Modern

| Aspect | Pre-Modern | Modern |
|--------|------------|--------|
| Focus | Desegregation | Educational opportunity |
| Collection | Paper/manual | Web-based |
| Variables | Limited | Extensive |
| Frequency | Irregular | Biennial |
| Coverage | Sample | Universe (from 2015) |

## Year-by-Year Changes

### 2011-12 Collection

**First modern CRDC**

| Aspect | Detail |
|--------|--------|
| Coverage | Sample (~7,000 LEAs, ~72,000 schools) |
| New elements | Restraint/seclusion, preschool discipline |
| Form structure | Redesigned school and LEA forms |
| Submission | Web-based system |

**Key data elements introduced**:
- Restraint and seclusion
- Preschool suspensions and expulsions
- Harassment/bullying categories
- Advanced course access
- Staff qualifications

### 2013-14 Collection

**Expanded sample**

| Aspect | Detail |
|--------|--------|
| Coverage | Larger sample (~16,500 LEAs) |
| Changes | Minor refinements to definitions |
| New elements | Some optional elements added |

**Key changes**:
- Expanded sample size
- Clarified definitions based on 2011-12 experience
- Added optional elements for certain data

### 2015-16 Collection

**Near-universe collection**

| Aspect | Detail |
|--------|--------|
| Coverage | Near-universe (~96,000 schools) |
| New elements | Chronic absenteeism, preschool expansion |
| Significance | First near-complete national data |

**Major additions**:
- **Chronic absenteeism** - First collection of this variable
- Expanded preschool data
- Sexual harassment of students
- Sports participation (Title IX)

**What this means**:
- First year suitable for national totals
- Baseline for time series
- Comprehensive school-level coverage

### 2017-18 Collection

**Full universe, major expansion**

| Aspect | Detail |
|--------|--------|
| Coverage | Full universe (~96,000 schools) |
| New elements | Computer science, expanded staffing |
| Puerto Rico | First inclusion |

**Major additions**:
- **Computer science offerings and enrollment**
- School resource officer data
- Teacher absenteeism data
- Puerto Rico schools (first time)

**Expanded elements**:
- More detailed staffing variables
- Additional course offerings
- Refined harassment categories

### 2020-21 Collection

**COVID-impacted year**

| Aspect | Detail |
|--------|--------|
| Coverage | Universe (~97,500 schools) |
| Special context | COVID-19 pandemic |
| New elements | COVID-specific variables |

**COVID-specific additions**:
- Remote instruction amount
- Student remote learning percentage
- Internet and device access

**Important notes**:
- Data reflects pandemic disruption
- Many schools operated remotely
- Discipline data artificially low
- **NOT COMPARABLE** to other years for trends

**New variables (permanent)**:
- Suspension instances (total count)
- School days missed due to suspension

### 2021-22 Collection

**Post-pandemic baseline**

| Aspect | Detail |
|--------|--------|
| Coverage | Universe (~98,000 schools) |
| Context | Return to in-person learning |
| Changes | COVID variables retained |

**Significance**:
- First "normal" year post-pandemic
- Better baseline for future comparisons
- COVID variables included for transition

### 2023-24 Collection

**Current collection**

| Aspect | Detail |
|--------|--------|
| Status | In progress |
| Expected changes | Minor refinements |
| COVID variables | Likely reduced |

## Definition Changes

### Suspension Definition Evolution

| Year | Definition |
|------|------------|
| 2011-14 | Students receiving one or more out-of-school suspensions |
| 2015+ | Separate counts for "one" and "more than one" OSS |

### Harassment Categories

| Year | Categories |
|------|------------|
| 2011-14 | Race, sex, disability (3 categories) |
| 2015+ | Added sexual orientation, religion (5 categories) |

### Chronic Absenteeism

| Year | Definition |
|------|------------|
| Pre-2015 | Not collected |
| 2015+ | Students missing 15 or more school days |
| 2020-21 | Same definition but complicated by remote learning |

### Restraint Definition Clarification

| Year | Change |
|------|--------|
| 2011-14 | Initial definitions |
| 2015+ | Clarified exclusion of physical escort |
| 2017+ | Additional guidance on timeout vs. seclusion |

## Coverage Evolution

### Sample to Universe Transition

```
2011    Sample (~7,000 LEAs)
        └── Stratified sample, not generalizable
        
2013    Expanded Sample (~16,500 LEAs)
        └── Larger but still sampled
        
2015    Near-Universe (~96,000 schools)
        └── First comprehensive coverage
        
2017+   Full Universe
        └── All public schools required
```

### Geographic Coverage

| Year | States | Territories |
|------|--------|-------------|
| 2011-15 | 50 states + DC | None |
| 2017+ | 50 states + DC | Puerto Rico |

### School Types Covered

|-------------|----------|
| Traditional public | All years |
| Charter | All years |
| Magnet | All years |
| Alternative | All years |
| Virtual | 2015+ (expanded) |
| Justice facility | All years |

## Implications for Analysis

### Time Series Recommendations

| Analysis Period | Recommendation |
|-----------------|----------------|
| 2011-2013 only | Can compare, but both sampled |
| 2011-2017 | Not recommended (sample vs. universe) |
| 2015-2017 only | Good for trends |
| 2015-2020 | Good, but note COVID |
| Including 2020-21 | Not recommended for trends |
| 2017-2021-22 | Good post-pandemic baseline |

### Variable Availability Matrix

```python
def check_variable_availability(variable, years):
    """
    Check which years a variable is available.
    """
    availability = {
        'chronic_absenteeism': [2015, 2017, 2020, 2021],
        'computer_science': [2017, 2020, 2021],
        'suspension_instances': [2020, 2021],
        'teacher_absenteeism': [2017, 2020, 2021],
        'oss_total': [2011, 2013, 2015, 2017, 2020, 2021],
        'restraint_physical': [2011, 2013, 2015, 2017, 2020, 2021],
    }
    
    var_years = availability.get(variable, [])
    return {
        'available_years': [y for y in years if y in var_years],
        'missing_years': [y for y in years if y not in var_years]
    }
```

### Comparing Across Years

**Safe Comparisons**:
- Same variable, same years (2015+)
- Relative measures (disparity ratios)
- Within-school changes over time

**Problematic Comparisons**:
- Pre-2015 vs. post-2015 (coverage change)
- Variables that changed definition
- Including 2020-21 in trends
- Variables not collected all years

### Documentation Requirements

When using CRDC data across years, document:

1. **Years used** - Which collection years
2. **Coverage type** - Sample vs. universe
3. **Variable definitions** - For each year if changed
4. **Missing years** - If variable not available
5. **COVID exclusion** - If excluding 2020-21
6. **School panel** - If restricting to consistent schools

### Where to Find Historical Documentation

- **ED.gov** - Official CRDC page, historical documents
- **CRDC Resource Center** - Collection materials
- **Education Data Portal mirrors** - Variable documentation via codebook `.xls` files (use `get_codebook_url()` from `fetch-patterns.md`)
- **NCES** - CRDC partnership documentation
- **Federal Register** - OMB clearance documents

> **Note:** When comparing variables across collection years, always verify definitions against the codebook for each specific year. The Portal codebook reflects the most recent collection; for earlier years, consult OCR's official data element lists.

### Version Tracking

Each CRDC collection has:
- OMB control number: 1870-0504
- Data element versions documented
- Form revision histories
- Public comment documentation
