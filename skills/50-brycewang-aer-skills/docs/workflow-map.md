## When to Loop

- **Identification rebuild** triggered by an R&R targeting the design вҶ’ loop back to `aer-identification` then forward
- **Format-only revisions** вҶ’ skip back only to `aer-tables-figures` or `aer-submission`
- **Venue change** after rejection вҶ’ `aer-topic-selection` again, then `aer-introduction` for re-framing

## The Router

`aer-workflow` is the entry point when the user is unsure where they are. It does not perform work вҖ” it picks the next skill.
